# Student API (Spring Boot + MongoDB)
Project Overview

This is a RESTful API built using Spring Boot and MongoDB to manage student data.

> Tech Stack
Java
Spring Boot
MongoDB
Maven

> Features
Create Student
Get All Students
Get Student by ID
Update Student
Delete Student

> API Endpoints
Method	Endpoint	Description
POST	/students	Create student
GET	/students	Get all students
GET	/students/{id}	Get by ID
PUT	/students/{id}	Update
DELETE	/students/{id}	Delete

> Sample JSON
{
  "name": "John Doe",
  "email": "john@example.com",
  "age": 21
}
